## Community highlights

With nearly 60 participants, this year brought more breadth than ever before. People worked across web development, hardware projects, art, and half-baked ideas that finally got the attention they deserved.

### Verilog to Minecraft Compiler

If you have ever worked with an FPGA before, you are likely familiar with [Verilog](https://en.wikipedia.org/wiki/Verilog). It is
a language used to create circuits and processors. If you have played Minecraft, you might be familiar with Redstone. DitrusNight
was able to create a Verilog to Minecraft Redstone compiler where you give it Verilog and it outputs a corresponding Minecraft
Redstone circuit. They were able to make functional RAM and ROM components with this project.

### 2D Printer

Most people are familiar with 3D printers. Imagine if we had a 3D printer but it was only 2 dimensions. [Madhav](https://github.com/MadhavVP)
made [just that](https://github.com/MadhavVP/2D-Printer) by taking 3D printer parts and removing the vertical axis.
Instead of the traditional way that printers works, it uses a pen to draw the images.

### Calculator Programming Language

We can't forget the projects that teach you fundamental CS concepts. [Daniel](https://github.com/ninebitcomputer)
made [a calculator language in Rust](https://github.com/ninebitcomputer/ro). Creating a programming language is not a trivial
task and even just turning the code into something the computer can work with requires a lot of learning.
